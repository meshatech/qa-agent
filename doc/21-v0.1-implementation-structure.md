# 21 — Estrutura de implementação v0.1

## Objetivo

Transformar as specs em um MVP executável em TypeScript/NestJS, mantendo o runtime determinístico e a LLM confinada à decisão de próxima ação.

## Stack v0.1

```txt
Node.js 20+
TypeScript
NestJS
Playwright
Zod
LangChain JS
OpenAI provider inicial
Vitest ou Jest
```

## Definição de pronto da v0.1

```txt
1. Executa uma demanda simples em app local/fixture
2. Usa OpenAI via LangChainDecisionProvider
3. Mantém contratos Zod como fonte da verdade
4. Gera run directory completo
5. Bloqueia stale observation
6. Não reutiliza el_* antigo em validação
7. Resolve dados dinâmicos por RunDataStore
8. Aguarda quiescência após toda ação
9. Recovery exausto gera evidência sanitizada
10. CLI retorna exit code correto
```
